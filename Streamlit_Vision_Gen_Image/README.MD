## ğŸ“¦ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. **Create a virtual environment (optional but recommended)**
   ```bash
   python -m venv .venv
   ```
